3D_Viewer

About programm

Программа предназначена для визуализации .obj файлов, а именно вершин и рёбер.

How to start

Для того что бы отобразить obj файл, необходимо указать программе путь до него, для этого на графическом интерфейсе нужно нажать кнопку "Open file", после чего в появившемся окне выбрать указать директорию и сам obj файл. Если все указано верно, то программа отобразит модель. 

Далее можно настроить отображение модели, для этого есть групп кнопок "Move", "Rotation", "Scale"

Управление положением модели на экране осуществляется нажатием групп кнопок Move, Rotate и Scale для перемещения, вращения и мастшабиированния модели соответственно,  измененеие отображения настроивается по осям X, Y, Z.

Группа Back color позволяет поменять цвет фона объекта, для этого необходимо нажать кнопку "Change color", под кнопкой будет отображаться текущий выбранный цвет фона, после нажатия "OK" он применится к основному окну отображения.

Группа "Line color " позволяет менять цвет линий отображаемого объекта, логика идентична Backround.

Группа "Vertexes color " позволяет менять цвет линий отображаемого объекта, логика идентична Backround.

Группа "Vertexes form " позволяет установить в вершины объекта круглые или квадратные поинты. Для того, чтобы выбрать поинт нажмите на ^.

Для того, чтобы помеять толщину линии выберите нужный размер в " Line width " и нажмите кнопку " Apply ", аналогично с размером вершины

Когда настроено отображение модели, можно сделать скриншот готовой "отрендереной" модели для этого необходимо нажать кнопку "Screenshot",в открывшемся окне выбрать путь сохранения изображения, в случае невозможности создания скриншота будет выведено сообщение, если скриншот сохранен, то никакого сообщения не будет.
